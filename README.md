# Create Missiles

![icon_256](<https://raw.githubusercontent.com/Woukie/create-missiles/refs/heads/1.20.1/icons/icon_256.gif>)

Minecraft missle mod for Forge and Fabric, depends on Create.

## Features

Subject to change.

- Launch long-range missiles
- Mix and match unique warheads, chassis and thrusters
- Each assembly has a unique model that depends on its build stage
- The navigation panel lets you click on a map to set the target and view a simulated trajectory that gets as close as possible to the target
- Set the burn time to change the height of the trajectory
- Fire survey drones to create a map of an area
- Integration with some other mods
